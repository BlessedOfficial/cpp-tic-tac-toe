# 🎮 C++ Tic Tac Toe

A C++ implementation of the classic Tic Tac Toe game, showcasing problem-solving, game logic, input validation, and structured code organization.

## 📌 About The Project

This project is a console-based Tic Tac Toe game built from scratch using C++. It was created to strengthen my understanding of core programming concepts, including functions, arrays, loops, conditional logic, and game state management.

The project will continue to evolve as I explore more advanced C++ concepts, including object-oriented programming, optimization, and graphical user interfaces.

## ✨ Features

- Two-player gameplay
- Player marker selection (X/O)
- Turn management system
- Valid move checking
- Prevention of occupied spaces
- Win detection
  - Rows
  - Columns
  - Diagonals
- Console-based game board
